# 通讯大师 (com-master) 

一个基于Qt6的串口通信工具，集成了串口终端、串口收发器、IEC61850客户端、SSH客户端和网络工具功能，专为电子设备测试和通信协议分析而设计。

## 功能特性

### 串口终端 (Serial Terminal)
- 支持多种波特率、数据位、校验位和停止位配置
- 终端式界面，支持命令行交互
- 右键菜单支持复制、粘贴和清空操作
- 实时显示串口数据

### 串口收发器 (Serial Transceiver)
- 十六进制和ASCII格式发送/接收
- 自动发送功能，可配置发送间隔
- 帧计数和字节统计
- 时间戳显示选项
- 自定义帧分隔符
- 数据验证功能

### IEC61850客户端 (IEC61850 Client)
- 完整的IEC61850协议栈支持
- 数据模型浏览和导航
- 数据读写操作
- 数据集管理
- 报告控制
- 控制服务

### SSH客户端 (SSH Client)
- 支持SSH协议远程连接
- 完整的终端模拟器功能
- 支持密码认证和公钥认证
- 交互式用户名和密码输入
- 连接状态实时显示
- 主机密钥验证

### 网络工具 (Network Tool)
- **TCP服务器**
  - 支持多客户端连接
  - 客户端列表显示和管理
  - 广播发送和定向发送功能
  - 自动发送功能
  - 实时连接状态监控
- **TCP客户端**
  - 支持连接到远程TCP服务器
  - 自动发送功能
  - 连接状态实时显示
- **UDP**
  - 支持UDP服务器模式
  - 支持发送到指定远程端点
  - 最近端点记录
  - 自动发送功能
- **通用功能**
  - 十六进制和ASCII格式发送/接收
  - 数据校验功能（CRC8、CRC16、CRC32、累加和、异或和等）
  - 自动发送，可配置发送周期
  - 时间戳显示选项
  - 日志记录和保存功能
  - 显示发送数据选项

## 系统要求

- **操作系统**: Linux (支持多架构: amd64, arm64, loong64, riscv64)
- **Qt版本**: Qt 6.x
- **编译器**: 支持C++17标准的编译器
- **依赖库**:
  - qt6-base-dev
  - qt6-l10n-tools
  - qt6-tools-dev
  - qt6-tools-dev-tools
  - qt6-svg-dev
  - qt6-serialport-dev
  - libssh-dev

## 安装方法

### 从源码编译

```bash
git clone https://github.com/lhdjply/com-master.git
cd com-master
mkdir build && cd build
cmake ..
make -j$(nproc)
sudo make install
```

### 使用Debian包

从[Releases](https://github.com/lhdjply/com-master/releases)页面下载适合您架构的.deb包，然后安装：

```bash
sudo dpkg -i com-master_*.deb
```

## 使用说明

### 串口终端
1. 启动应用程序
2. 选择"串口终端"标签页
3. 配置串口参数（端口、波特率等）
4. 点击"打开"按钮连接串口
5. 在终端中输入命令，按回车发送

### 串口收发器
1. 选择"串口收发器"标签页
2. 配置串口参数并连接
3. 在发送区域输入数据，可选择十六进制格式
4. 点击"发送"按钮或启用自动发送
5. 接收区域将显示收到的数据

### IEC61850客户端
1. 选择"IEC61850客户端"标签页
2. 点击连接按钮，输入服务器地址和端口
3. 连接成功后，浏览左侧的数据模型树
4. 双击数据对象查看详情
5. 使用读取/写入按钮进行数据操作

### SSH客户端
1. 选择"SSH客户端"标签页
2. 输入主机地址、端口号（默认22）、用户名和密码
3. 点击"连接"按钮建立SSH连接
4. 如果未提供用户名和密码，可在终端中交互式输入
5. 连接成功后，在终端中进行远程操作
6. 点击"断开"按钮结束连接

### 网络工具
1. 选择"网络工具"标签页
2. 在协议下拉框中选择TCP服务器、TCP客户端或UDP模式

**TCP服务器模式：**
1. 配置监听端口
2. 点击"启动"按钮开始监听
3. 等待客户端连接，客户端列表会显示已连接的客户端
4. 在发送区域输入数据，可选择十六进制格式
5. 使用"广播"按钮发送给所有客户端，或选中客户端后使用"发送到选中"按钮
6. 可启用自动发送功能并设置发送周期
7. 点击"停止"按钮关闭服务器

**TCP客户端模式：**
1. 输入服务器IP地址和端口
2. 点击"连接"按钮连接到服务器
3. 连接成功后，在发送区域输入数据
4. 点击"发送"按钮发送数据
5. 可启用自动发送功能并设置发送周期
6. 点击"断开"按钮断开连接

**UDP模式：**
1. 配置本地监听端口
2. 点击"启动"按钮开始监听
3. 输入目标IP地址和端口
4. 在发送区域输入数据
5. 点击"发送"按钮发送数据到目标端点
6. 可启用自动发送功能并设置发送周期
7. 最近使用的端点会记录在列表中，方便快速选择
8. 点击"停止"按钮关闭UDP服务

**通用功能：**
- 使用"校验"按钮为数据添加校验码（CRC8、CRC16、CRC32、累加和、异或和等）
- 使用"清除发送区"按钮清空发送区域
- 接收区域显示收到的数据，可勾选"16进制"和"显示时间"选项
- 日志区域记录操作日志，可清空或保存日志

### 项目结构
```

├── src/                    # 主程序源码
```
